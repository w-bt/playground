# Problem: Add Binary

Given two binary strings `a` and `b`, return their sum as a **binary string**.

---

## Examples

- **Example 1**  
  **Input:** `a = "11"`, `b = "1"`  
  **Output:** `"100"`

- **Example 2**  
  **Input:** `a = "1010"`, `b = "1011"`  
  **Output:** `"10101"`

---

## Constraints
- `1 <= len(a), len(b) <= 10^4`  
- `a` and `b` consist only of characters `'0'` and `'1'`.  
- Strings may be long; avoid integer conversion that can overflow.
